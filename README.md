# Send-Emails
Send Email Using Xampp Local Server in PHP

## Requirements
* XAMPP >= 7.0
* PHP >= 7.1

## Configure XAMPP To Send Mail From Localhost
* (1) Open Xampp Folder Form Your Device & Go " Xampp/sendmail/sendmail.ini ".
